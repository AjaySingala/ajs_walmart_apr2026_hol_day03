Context Window:
Context window size is 8000 tokens
System prompt: 	200 tokens
User query:	   	100 tokens
-------------------------
				300 tokens
For the AI assistant to respond, it has 8000 - 300 tokens available: 7700 tokens

System prompt: 	200 tokens
User query:	   	100 tokens
Document:	   3000 tokens
-------------------------
			   3300 tokens
For the AI assistant to respond, it has 8000 - 3300 tokens available: 4700 tokens

System prompt: 	200 tokens
User query:	   	100 tokens
Document:	   3000 tokens
More context:  1000 tokens
-------------------------
			   4300 tokens
For the AI assistant to respond, it has 8000 - 4300 tokens available: 3700 tokens

Context window is made up of the:
- system prompt
- user query
- docs (RAG)
- other context
- output (from the AI model)

System prompt: 	200 tokens
User query:	   	100 tokens
Document:	   5000 tokens
More context:  1000 tokens
-------------------------
			   6300 tokens (input)
For the AI assistant to respond, it has 8000 - 6300 tokens available: 1700 tokens.
If the AI response is > the available no. of tokens (1700), the output will be truncated.

Fine-tuning:
Niche process - trade secret.
Use AI (LLM) to speed up the process of analysing company data to come up with decisions.

ChatGpt (public)
Create their own LLM specific to their work/process/data.
Creating an LLM Scratch : very time consuming, very expensive, very talented and expert people.

They took an existing LLM and "re-trained" it on their data/process.
- Fine-tuning
